welcome to Code Girl

We are a non-profit orgnization that provides affordable program for ladies to learn web and software development there by creating a sustainable goal and breaching the gap between inequalities in the tech industry in Africa

We’re changing tech

Women have made up less than 25 percent of the hires in tech over the past 20 years. That needs to change. Code Girl is building a diverse and inclusive tech workforce — one you’d be proud to be a part of. There’s room for all of us in tech. Join us!
